Last updated: March 22, 2026

This privacy policy applies to the ISP Network Tool app (the "Application") created by Md Mahmudul Hasan Rudra (the "Service Provider") as a free service. This service is provided "AS IS".

1. Information Collection and Use

The Application collects limited information necessary to provide network diagnostic and performance monitoring features.

  • Network Information: The app accesses network state, Wi-Fi status, and internet connection.

The Application does NOT collect personal data such as name, email, or phone number.

2. Location Information

The Application does not collect or store precise location data.

However, Android may require location permission to access Wi-Fi information. This is used only for network functionality, not tracking.

3. Third-Party Access

The Service Provider does not sell user data.

4. Data Retention

Data is retained only as long as necessary for app functionality.

You can stop all data collection by uninstalling the Application.

5. Children's Privacy

The Application is not intended for children under 13. No personal data is knowingly collected from children.
