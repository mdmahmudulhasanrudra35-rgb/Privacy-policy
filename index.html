Privacy Policy

Last Updated: February 7, 2026

This Privacy Policy explains how ISP Network Tool ("the App") handles user information.

No Data Collection

ISP Network Tool is designed as a user-centric educational application that does NOT collect, store, or transmit any personal information, usage data, or analytics.

Our Privacy Commitment

ISP Network Tool is built with privacy as a core principle. We believe in providing valuable networking tools without compromising user privacy.

Important: This application does not collect, store, or transmit any personal data, usage statistics, or analytics. All network testing and calculations happen locally on your device.

What We Don't Do

  • We do NOT collect any personal information
  • We do NOT track user activity or behavior
  • We do NOT save any data from the application
  • We do NOT sell anything (the app is free)
  • We do NOT use analytics or tracking services
  • We do NOT transmit any data to external servers

Internet Permission

The app requires INTERNET permission solely for:

  • Fetching and displaying your public IP address
  • Performing network tests (ping, latency, BDIX connectivity)
  • Accessing network information for educational purposes

Important: These network operations happen in real-time and no results are stored or transmitted externally.

Application Purpose

ISP Network Tool is designed as an educational tool for:

  • Teaching networking concepts to newcomers
  • Demonstrating routing principles
  • Making network engineering interesting and accessible
  • Providing practical ISP tools (bandwidth planning, BDIX testing, ping tools)

All calculations, tests, and results remain on your device and are not shared with anyone.

Local Processing Only

All features of ISP Network Tool work through local processing on your device:

  • Bandwidth Capacity Planning: Calculations are performed locally based on your inputs
  • BDIX Dashboard: Connectivity tests run in real-time with no data storage
  • Ping & Latency Tools: Results are displayed immediately and not saved
  • Advanced Analytics: All calculations happen on your device

Third-Party Services

ISP Network Tool does not integrate with any third-party services that collect user data. The app is completely standalone.

Changes to This Policy

If we ever make changes to this privacy policy (such as adding optional features), we will clearly notify users and update this page with a new "Last Updated" date.

Our commitment to privacy means we would never silently add data collection features.
